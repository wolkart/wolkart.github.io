Какое последнее значение выведет этот код? Почему?
		let i = 3;
		while (i) {
		  alert( i-- );
		}

		Ответ: 1.
		Каждое выполнение цикла уменьшает i. 
		Проверка остановит цикл при i = 0, т.к. 0 дает false.
	
		Какие значения выведет цикл while?
		Префиксный вариант ++i:
		let i = 0;
		while (++i < 5) alert( i );

		Постфиксный вариант i++:
		let i = 0;
		while (i++ < 5) alert( i );

		Ответы:
		От 1 до 4, т.к. операция ++i сначала увеличит i, а потом уже произойдёт сравнение
		От 1 до 5, т.к. оператор i++ увеличивает i, возвращая старое значение.
		Для каждого значения сначала происходит сравнение, а потом – увеличение.
	
		Какие значения выведет цикл for?
		Постфиксная форма:
		for (let i = 0; i < 5; i++) alert( i );

		Префиксная форма:
		for (let i = 0; i < 5; ++i) alert( i );

		Ответ: от 0 до 4 в обоих случаях.
